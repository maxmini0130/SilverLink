# SilverLink 관계 모델 정의서

- 목적: 관심 / 1촌 / 대화 개념을 팀 내에서 한 문장으로 통일하고, 코드/RLS/UX 문구의 기준으로 사용
- 기준일: 2026-04-15

---

## 핵심 한 줄 정의

| 개념 | 한 줄 정의 |
|------|-----------|
| **관심** | "이 사람이 마음에 들어요" — 가볍게 신호를 보내는 첫 걸음 |
| **1촌** | "서로 연결된 사이예요" — 양방향으로 승인된 관계 |
| **대화** | "조건이 맞으면 대화할 수 있어요" — 제한된 조건 하에서의 안전한 소통 |
| **같은 모임** | "같은 공간에 함께 있어요" — 오프라인 연결의 온라인 교두보 |

---

## 관계 단계별 상세 정의

### 1단계: 관심 (Interest)

- **정의**: 한 사람이 다른 사람에게 보내는 일방향 신호
- **조건**: 아무 조건 없이 보낼 수 있음 (차단 관계 제외)
- **효과**:
  - 상대방 알림에 표시됨
  - 상대방이 관심을 돌려보내면 "상호 관심" 상태가 됨
  - 단방향 관심 상태에서 상대의 `interested` 공개 피드를 볼 수 있음
- **취소**: 언제든지 취소 가능
- **DB 테이블**: `relationship_requests`

---

### 2단계: 상호 관심 (Mutual Interest)

- **정의**: 두 사람이 서로에게 관심을 보낸 상태
- **조건**: 내가 보낸 관심 + 상대가 나에게 관심을 보낸 상태
- **효과**:
  - 대화 시작 가능
  - 1촌 맺기 가능
  - 상대의 `interested` 공개 피드 열람 가능
- **DB**: `relationship_requests` 양방향 존재로 판단

---

### 3단계: 1촌 (Friend)

- **정의**: 상호 관심 또는 같은 모임 참여를 통해 정식으로 연결된 관계
- **조건** (둘 중 하나):
  - 상호 관심 상태에서 한쪽이 1촌 신청
  - 같은 모임에 함께 참여 중인 상태에서 1촌 신청
- **효과**:
  - 대화 시작 가능
  - 상대의 `friends` 공개 피드 열람 가능
  - 관계 목록에 표시됨
- **취소**: 현재 미구현 (추후 추가 예정)
- **DB 테이블**: `friendships` (user_low_id, user_high_id)

---

### 같은 모임 (Same Group)

- **정의**: 두 사람이 동일한 모임에 참여 중인 상태
- **조건**: `group_members` 테이블에 같은 `group_id`로 둘 다 존재
- **효과**:
  - 대화 시작 가능
  - 1촌 맺기 가능
  - 상대의 `same_group` 공개 피드 열람 가능
- **DB 테이블**: `group_members`

---

## 대화 가능 조건 요약

대화는 아래 세 가지 조건 중 **하나라도** 충족하면 시작할 수 있습니다.

| 조건 | 설명 |
|------|------|
| 상호 관심 | 서로 관심을 보낸 상태 |
| 1촌 | 정식으로 관계가 맺어진 상태 |
| 같은 모임 | 동일한 모임에 함께 참여 중 |

> 차단 관계가 있으면 어떤 조건을 만족해도 대화 불가.

---

## 공개범위와 관계 연결

| 공개범위 | 볼 수 있는 사람 |
|----------|----------------|
| 나만 보기 | 본인만 |
| 1촌만 보기 | 본인 + 1촌 |
| 관심 있는 사람만 보기 | 본인 + 상호 관심 또는 1촌 관계 |
| 같은 모임 사람만 보기 | 본인 + 같은 모임 참여자 |
| 전체 인증회원 보기 | 모든 로그인 사용자 |

---

## 관계 흐름도

```
처음 만남
    ↓
관심 보내기 (단방향)
    ↓
상대도 관심 보내기 → 상호 관심
    ↓
1촌 맺기 또는 같은 모임 참여
    ↓
대화 시작 가능
    ↓
오프라인 모임 연결 (후속)
```

## 코드/UX 용어 기준

| 개념 | 코드 내 값 | 화면 표시 문구 |
|------|-----------|--------------|
| 관심 보내기 | `relationship_requests` INSERT | 관심 보내기 |
| 관심 취소 | `relationship_requests` DELETE | 관심 취소 |
| 상호 관심 | sentInterest && receivedInterest | 상호 관심 |
| 1촌 맺기 | `friendships` INSERT | 1촌 맺기 |
| 1촌 | `friendships` 존재 | 1촌 |
| 대화 시작 | `conversations` + `conversation_members` INSERT | 대화 시작 |
| 같은 모임 | `group_members` 공유 | 같은 모임 N개 |
