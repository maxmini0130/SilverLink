- 테스트 환경: 브라우저 직접 수동 테스트 기준

---

## 테스트 규칙

- 각 항목을 직접 실행하여 통과 여부를 `[ ]` → `[x]`로 표시
- FAIL 시 해당 항목 옆에 `❌ 증상 메모` 추가
- 테스트 완료 후 날짜와 결과를 하단 이력에 기록
