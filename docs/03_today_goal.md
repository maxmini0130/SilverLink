# 오늘 작업 목표

## 오늘 목표
로그인 후 사용자가 처음 들어왔을 때 이동할 프로필 입력 화면의 skeleton을 만든다.

## 완료 기준
1. 로그인 상태 사용자가 접근 가능한 프로필 입력 페이지가 있다.
2. 닉네임, 성별, 연령대, 지역, 관심사 입력 UI가 있다.
3. 저장 버튼이 있다.
4. 아직 실제 저장 로직이 완전하지 않아도 괜찮지만, 구조는 분리되어 있어야 한다.
5. 기존 로그인 페이지와 리다이렉트 흐름을 깨지 않는다.

## 오늘 하지 않을 것
- 채팅 구현
- 추천 알고리즘 구현
- 소모임 구현
- 관리자 구현
- 결제 / 광고 / 건강 기능

## 개발 원칙
- 기존 프로젝트 구조를 우선 존중
- 한 번에 큰 기능을 넣지 않기
- 기존 auth 흐름 깨지 않기
- UI는 단순하고 큰 글자 중심
