# SeniorLink 서비스 정의

## 서비스 한 줄 정의
SeniorLink는 50~70대가 지역과 관심사를 기반으로 새로운 친구를 찾고, 안전하게 대화하고, 소모임에 참여할 수 있도록 돕는 시니어 소셜 플랫폼이다.

## 핵심 타깃
- 1차 타깃: 스마트폰 사용이 가능한 50~60대 액티브 시니어
- 2차 타깃: 취미/대화/지역 모임 니즈가 있는 60대 후반 이상 사용자

## 해결하려는 문제
1. 외로움과 사회적 단절
2. 새로운 인간관계를 만들 기회의 부족
3. 시니어 친화적인 디지털 커뮤니티의 부족

## 핵심 가치
1. 쉬운 사용성
2. 관심사/지역 기반 연결
3. 안전한 소통 구조

## MVP 핵심 흐름
로그인 → 프로필 입력 → 관심사/지역 설정 → 추천 친구 보기 → 채팅 시작 → 소모임 탐색
