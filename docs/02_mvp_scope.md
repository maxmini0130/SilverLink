# SeniorLink MVP 범위

## MVP 포함
- 회원가입 / 로그인
- 프로필 입력 / 수정
- 관심사 / 지역 설정
- 추천 친구 리스트
- 1:1 채팅
- 소모임 목록 / 상세 / 참여 / 개설 기본
- 신고 / 차단
- 관리자 최소 기능

## MVP 제외
- 음성 메시지
- 영상 / 라이브 기능
- 건강 알림 / 복약 기능
- 결제 / 유료 멤버십
- 광고 시스템
- 가족 보호자 SOS
- 고급 AI 추천
- 고도화된 인증 기능

## 제품 원칙
- 큰 글자와 단순한 화면
- 한 화면 한 목적
- 기능 추가보다 흐름 완성 우선
